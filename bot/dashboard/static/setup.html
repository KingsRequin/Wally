🔗 Connexion Discord

Crée une application sur discord.com/developers si ce n'est pas fait.

💡 Applications → ton app → onglet Bot → bouton Reset Token → copie le token
💡 Dans Discord : Paramètres → Avancés → active Mode développeur. Puis clic droit sur ton serveur → Copier l'identifiant
💡 Applications → ton app → onglet OAuth2 → Client ID
💡 Même page OAuth2 → clique sur Reset Secret

🔑 Clés API

Ces clés permettent à ton bot d'utiliser l'intelligence artificielle.

💡 Va sur platform.openai.com/api-keys → Create new secret key → copie la clé (commence par sk-)
Uniquement si tu veux utiliser Claude. Va sur console.anthropic.com → API Keys. Laisse vide sinon.
Permet la recherche web. Va sur app.tavily.com. Laisse vide sinon.
